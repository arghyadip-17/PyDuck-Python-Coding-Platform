# 🦆 PyDuck – Python Coding Platform
## 📌 Overview
PyDuck is a lightweight Python coding practice platform inspired by competitive programming sites.
It allows users to solve beginner-friendly programming problems directly in the browser, run their Python code, and get instant feedback based on predefined test cases.
The platform is built using Flask and focuses on simplicity, clean UI, and core problem-solving skills.
## 🎯 Key Features
- 🧠 Collection of beginner Python problems
- 🧪 Automatic test case evaluation
- ⚡ Instant feedback (Accepted / Wrong Answer / Runtime Error)
- 🌙 Light & Dark mode support
- 🧩 Clean and minimal coding interface
- 🔐 Safe execution using temporary files
- 🧑‍💻 No login required

## 🧠 How It Works
1. User selects a problem from the sidebar
2. Writes Python code in the editor
3. Code is executed in a temporary isolated file
4. Output is matched against predefined test cases
5. Result is displayed instantly
